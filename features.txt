ResumeCraft - Implemented Features

## Core Features
1. **Personal Information**: 
   - First & Last Name
   - Contact Info (Email, Phone with Country Code Picker)
   - Location
   - LinkedIn Profile
   - GitHub Profile

2. **Education Section**:
   - Categorized entries (Class X (10th), Class XII (12th), Diploma, Degree, Post Graduate)
   - Score/Percentage/CGPA tracking

3. **Skills Management**:
   - Categorized selection interface (Frontend, Backend, Database, Mobile, DevOps, Tools, Languages, Soft Skills)
   - Custom skill addition

4. **Experience Tracking**:
   - Job history with Title, Company, Duration
   - Bullet point descriptions

5. **Projects Portfolio**:
   - Project names
   - Technologies used
   - Descriptions

6. **Resume Preview & Analysis**:
   - Real-time preview of the resume
   - Editorial/Minimalist layout
   - Automated analysis (completeness, action verbs, required fields)

## Recent Enhancements (Persistence & UI)
7. **Persistence & History**:
   - Local storage of resumes using JSON format.
   - "My Resumes" dashboard on the Welcome screen.
   - Edit previously saved resumes with all fields preserved.
   - Swipe-to-delete functionality for easy management.

8. **Resume Renaming**:
   - Optional custom titles for each resume (e.g., "Software Engineer 2024").
   - Titles are displayed in the history list for easy identification.

9. **PDF Export & Sharing**:
   - Professional PDF generation using Apple's modern ImageRenderer.
   - Support for off-screen rendering (fixes empty PDF bug).
   - Custom filenames based on Resume Title.

10. **System Stability**:
    - Modular View Architecture (Resolved "Expression too complex" compiler errors).
    - Data Consistency: Fixed immutable property crashes and duplicate file creation.
    - Backward Compatibility: Gracefully handles older resume files without titles.

11. **Welcome Screen Polish**:
    - "ResumeCraft" branding
    - Premium Typography (Times New Roman)
    - Subtle "Breathing" animation for the logo
    - Haptic Feedback on interaction
    - Full Dark Mode support
