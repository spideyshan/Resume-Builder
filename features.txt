ResumeCraft - Implemented Features

## Core Features
1. **Personal Information**: 
   - First & Last Name
   - Contact Info (Email, Phone with Country Code Picker)
   - Location
   - LinkedIn Profile
   - GitHub Profile

2. **Education Section**:
   - Categorized entries (Class X (10th), Class XII (12th), Diploma, Degree, Post Graduate)
   - Score/Percentage/CGPA tracking

3. **Skills Management**:
   - Categorized selection interface (Frontend, Backend, Database, Mobile, DevOps, Tools, Languages, Soft Skills)
   - Custom skill addition

4. **Experience Tracking**:
   - Job history with Title, Company, Duration
   - Bullet point descriptions

5. **Projects Portfolio**:
   - Project names
   - Technologies used
   - Descriptions

6. **Resume Preview & Analysis**:
   - Real-time preview of the resume
   - Editorial/Minimalist layout
   - Automated analysis (completeness, action verbs, required fields)

## Recent Enhancements (Persistence & UI)

## AI & Smart Features (New)
12. **ATS Optimization**:
    - **Real-time Scoring**: Calculates an ATS Score (0-100) based on content depth, keywords, and structural completeness.
    - **Smart Analysis**: Uses Apple's **NaturalLanguage** framework to analyze tone and keyword usage.
    - **Actionable Feedback**: Suggests specific improvements like adding strong action verbs, expanding bullet points, and including essential links (LinkedIn/GitHub).

13. **Resume Parsing (Import)**:
    - **One-Click Import**: Automatically fill the resume form by uploading a PDF or Image.
    - **On-Device OCR**: Uses Apple's **Vision Framework** to scan text securely on the device.
    - **Smart Mapping**: Automatically detects Name, Email, Phone, LinkedIn, and sections like Experience/Education.
